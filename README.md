# Convertisseur de Température

Un programme Java simple qui convertit des températures entre **Celsius** et **Fahrenheit**.

## Ce que fait le programme

- Convertir une température de **Celsius vers Fahrenheit**
- Convertir une température de **Fahrenheit vers Celsius**
- Demande à l'utilisateur s'il veut effectuer une autre conversion

## Prérequis

- Java JDK installé sur votre machine

## Cloner le projet

```bash
git clone https://github.com/votre-username/apprendre_java.git
cd apprendre_java
```

## Exécuter le programme

**1 - Ouvrir avec intelliJ ou tout autre IDE java**
